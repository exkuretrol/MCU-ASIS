章節 7 MySQL 使用者管理

7.1 新增使用者與使用者資料庫

連線至 MySQL 的終端機。

sudo mysql

如果 root 的使用者帳號不是使用 auth_socket 的方式驗證的話，需要以帳號密碼登入。以下是登入用的指令

mysql -u root -p

創建以 密碼 驗證名為 使用者名稱 的使用者

create user '<使用者名稱>'@'localhost' identified by '<密碼>';`

創建以 `使用者名稱` 為名的資料庫

``` {.mysql .prefixed_mysql}
create database <使用者名稱> character set utf8mb4 collate utf8mb4_unicode_ci;

允許使用者 使用者名稱 的使用者擁有在資料庫 使用者名稱 的所有權限

grant all privileges on <使用者名稱>.* to '<使用者名稱>'@'localhost';

7.2 刪除使用者

下列指令可以刪除名為 使用者名稱 的使用者

drop user '<使用者名稱>'@'localhost';
